customIconproperty
customIcon:Bitmap  [read-write]

Custom icon which will be used with the badge. This is only nessacary if no icon has been defined within the application descriptor.

The default value is null.

labelproperty 
label:String  [read-write]

The label that shows up on the badge. Keep in mind that the space for your label is limited! If the label is null or empty no badge will be displayed.

The default value is null.
